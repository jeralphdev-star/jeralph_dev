## 🏆 Featured Projects

### 📁 [DormIT — Dormitory Management System](https://github.com/jeralphdev)
A full-stack web-based dormitory management system built for the CSUCC Ladies Dormitory.
Features include student application & DSS scoring, room assignment, payment tracking,
admin/cashier modules, email reminders, OTP-based forgot password, and Grad-CAM explainability.

**Technologies:** PHP, MySQL, JavaScript, HTML/CSS, XAMPP, PHPMailer

---

### 📁 [Fashion MNIST Image Classification](https://github.com/jeralphdev/Fashion-MNIST-Image-classification)
Neural network for classifying clothing items using TensorFlow and Keras.
Achieved 88.95% accuracy through optimizations including increased epochs,
more neurons, and additional hidden layers.

**Technologies:** Python, TensorFlow, Keras, NumPy, Matplotlib, Google Colab

---

### 📁 [CNN Image Classifier with Grad-CAM](https://github.com/jeralphdev)
Multi-class CNN image classifier with full evaluation pipeline including Confusion Matrix,
ROC/AUC, Precision/Recall/F1, and Grad-CAM explainability for visualizing model decisions.
Includes regularization techniques: Batch Normalization, Dropout, Data Augmentation, and Early Stopping.

**Technologies:** Python, TensorFlow, Keras, scikit-learn, OpenCV, Google Colab
